Cuso de Arduino básico

Linguagem de Programação C++

O C++ é uma linguagem de programação de nível médio, baseada na linguagem C. O desenvolvimento da linguagem começou na década de 80, por Bjarne Stroustrup.

O que é C++ e para que serve essa linguagem?

C++ é uma linguagem de programação open source orientada a objetos, utilizada tanto como linguagem de máquina quanto para softwares empresariais, acadêmicos, jogos, edição de imagens, fotos e vídeos.

Onde C++ é usado?

C++ é usado para o desenvolvimento de sistemas embarcados, bibliotecas gráficas, jogos, sistemas operacionais, entre outros.

O que é Arduimo?

Arduíno é uma placa de prototipagem eletrônica de código aberto. O projeto, surgido na cidade de Ivrea, na Itália, em 2005, inclui hardware e software livre e visa oferecer ferramentas adaptáveis e de baixo custo para a criação de projetos interativos de diversas ordens.

Faça você mesmo

A base do Arduíno é justamente a ideia do “faça você mesmo”, também conhecido pela sigla em inglês DIY (de do it yourself). Como existem placas de diferentes tamanhos, elas podem ser adaptadas aos mais variados projetos, como é o caso do uso do Arduíno Gemma, usado para a criação de um dispositivo de computação vestível.

Software

O Arduino Integrated Development Environment – ou Arduino Software (IDE) – contém um editor de texto para escrever código, uma área de mensagem, um console de texto, uma barra de ferramentas com botões para funções comuns e uma série de menus. Ele se conecta ao hardware Arduino e Genuino para fazer upload de programas e se comunicar com eles.

A IDE do Arduino também é open-source. Seu ambiente gráfico foi desenvolvido em Java e baseado em Processig e outras linguagens open-source. Para baixar essa linguagem, basta acessar a página oficial e fazer o download.
